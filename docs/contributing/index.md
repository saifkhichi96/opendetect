# Contributing

OpenDetect welcomes contributions from users and contributors across model quality, runtimes, tooling, and docs.

High-impact areas right now:

- hardware validation for Intel and AMD accelerators
- runtime compatibility notes (TensorRT/OpenVINO/ROCm/MIGraphX/DirectML)
- additional tutorials and end-to-end examples
- benchmark reproducibility scripts and environment metadata capture

See [Development Setup](development.md) for local setup.
